# 🍔 Food Delivery Time Prediction

### Machine Learning & Data Analytics Project

## 📌 Project Overview

This project predicts **food delivery time** using Machine Learning techniques.
It analyzes multiple factors such as **distance, weather conditions, traffic level, courier experience, preparation time, and vehicle type** to estimate delivery time.

The project demonstrates a complete **Data Analytics and Machine Learning pipeline**, including data analysis, visualization, feature engineering, model training, evaluation, and deployment using a Flask web application.

## 📊 Dataset Features

| Feature                | Description               |
| ---------------------- | ------------------------- |
| Distance_km            | Delivery distance         |
| Preparation_Time_min   | Food preparation time     |
| Courier_Experience_yrs | Courier experience        |
| Weather                | Weather condition         |
| Traffic_Level          | Traffic condition         |
| Time_of_Day            | Morning / Evening / Night |
| Vehicle_Type           | Bike / Scooter / Car      |
| Delivery_Time_min      | Target variable           |

## 📈 Model Performance

* **Mean Squared Error (MSE):** 68.44
* **R² Score:** 0.83
* **Adjusted R² Score:** 0.83

The model explains **83% variance in delivery time**, indicating good predictive performance.

```bash

```
